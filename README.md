# rag_chatbot: WhatsApp Document AI Assistant

A Retrieval-Augmented Generation (RAG) pipeline that brings intelligent document querying directly to WhatsApp.
Instead of navigating web interfaces, users can simply forward a document to a dedicated WhatsApp number and 
immediately start asking context-specific questions about its contents. 
Built with FastAPI, Twilio Sandbox, SQLAlchemy ORM, and the Gemini API, it provides low-latency,
accurate answers while calculating and managing context token expenditures behind the scenes.
